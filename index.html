CEVA LOGISTICS Agentic AI EU Pilot

IT RFQ requirement analysis

A two-agent system that automates how the IT presales team analyses customer tenders. Agent 1 ingests tender documents from SharePoint (Word, Excel, PDF, PowerPoint), extracts and structures the IT requirements against a checklist, flags gaps and generates clarification questions. Agent 2 takes the validated requirements and maps them to the right solution using decision trees maintained by the business, not in code. A human checkpoint sits between every stage, and no agent acts on its own. Built on the Mistral Workflows SDK with durable execution, so long multi-step runs survive restarts. Targeting an EU pilot.
